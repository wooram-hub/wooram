# GitHub Desktop 사용 가이드

## GitHub Desktop으로 푸시하기

### 1단계: GitHub Desktop에서 저장소 열기

1. **GitHub Desktop** 앱 실행
2. 메뉴에서 **File** → **Add Local Repository** 선택
3. **Choose...** 버튼 클릭
4. `C:\wooram` 폴더 선택
5. **Add repository** 클릭

### 2단계: GitHub에 로그인 (처음 사용 시)

1. GitHub Desktop에서 **File** → **Options** (또는 **Preferences**)
2. **Accounts** 탭 클릭
3. **Sign in** → **Sign in to GitHub.com** 클릭
4. 브라우저에서 GitHub 로그인 및 권한 승인

### 3단계: 푸시하기

1. 왼쪽 패널에서 변경사항 확인
   - `index.html`
   - `README.md`
2. 왼쪽 하단에 **Summary** 입력 (선택사항)
3. **Commit to main** 버튼 클릭
4. 상단 메뉴에서 **Repository** → **Push origin** 클릭
   또는 오른쪽 상단의 **Push origin** 버튼 클릭

### 4단계: 확인

GitHub 웹사이트에서 확인:
- https://github.com/wooram-hub/wooram
- Code 탭에서 파일들이 보이는지 확인

## 팁

- GitHub Desktop을 열어두면 자동으로 변경사항을 감지합니다
- 커밋 메시지를 입력하지 않아도 기본 메시지로 커밋됩니다
- **Sync** 버튼을 사용하면 Pull과 Push를 한 번에 할 수 있습니다
