# 매출 통계 대시보드

회사 매출을 월별, 주차별, 카테고리별로 집계하고 시각화하는 보고용 통계 프로그램입니다.

## 📋 주요 기능

### 1. 매출 데이터 업로드
- Excel 파일(.xls, .xlsx) 업로드
- 전자세금계산서 데이터 자동 파싱
- 날짜, 거래처, 금액 정보 자동 추출

### 2. 카테고리별 매출 구분
- **맑은이러닝**: 거래처명에 "맑은", "이러닝" 포함
- **콘텐츠**: 거래처명에 "콘텐츠" 포함
- **위캔디오**: 거래처명에 "위캔디오", "위캔", "디오" 포함

### 3. 월별/주차별 집계
- 월별 전체 매출 금액 표시
- 주차별 자동 구분 (1~5주차)
- 주차별 카테고리별 상세 내역 테이블

### 4. 전월/다음달 비교
- 전월 대비 증감률 표시
- 다음달 예상 금액 비교
- 증감률 자동 계산 및 표시

### 5. 시각화
- 주차별 매출 바 차트 (카테고리별 비교)
- 카테고리별 매출 분포 도넛 차트
- 반응형 디자인

### 6. 보고서 기능
- 보고서 작성 텍스트 입력
- 링크 공유 기능 (URL에 데이터 포함)
- PDF 출력 기능

## 🚀 사용 방법

### 1. 파일 업로드
1. 웹 페이지 열기
2. "📁 파일 업로드" 버튼 클릭
3. Excel 파일 선택 (전자세금계산서 목록)

### 2. 데이터 확인
- 자동으로 현재 월의 데이터가 표시됩니다
- 이전/다음 월 버튼(◀ ▶)으로 월 이동

### 3. 보고서 작성
- 하단 텍스트 영역에 보고 내용 작성
- "🔗 링크 공유" 버튼으로 공유 링크 생성
- "📄 PDF 출력" 버튼으로 PDF 다운로드

## 🛠 기술 스택

- **HTML5/CSS3**: 웹 인터페이스
- **JavaScript (Vanilla)**: 핵심 로직
- **Chart.js**: 데이터 시각화
- **SheetJS (xlsx.js)**: Excel 파일 파싱
- **jsPDF**: PDF 생성

## 📊 Excel 파일 형식

프로그램은 다음 컬럼을 자동으로 인식합니다:
- **날짜 컬럼**: "거래일자", "발행일자", "일자", "날짜" 등
- **거래처 컬럼**: "거래처", "업체", "회사", "상호" 등 (선택)
- **금액 컬럼**: "금액", "공급가액", "매출" 등

## 🎨 주요 화면 구성

1. **헤더**: 제목 및 액션 버튼
2. **월 선택기**: 현재 월 표시 및 전월/다음달 버튼
3. **비교 카드**: 전월/당월/다음달 매출 비교
4. **요약 카드**: 카테고리별 매출 요약 (3개)
5. **주차별 차트**: 주차별 매출 바 차트
6. **카테고리별 차트**: 도넛 차트
7. **상세 테이블**: 주차별 상세 내역
8. **보고서 작성**: 텍스트 입력 영역
